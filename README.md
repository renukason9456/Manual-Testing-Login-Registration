# Manual Testing – Login & Registration Module

## Project Overview
This project focuses on **manual testing** of the Login and Registration module of a web application.  
The goal is to ensure the functionality, validation, and reliability of user authentication features.

## Application Under Test
This project was executed on a publicly available demo website for practicing manual testing:  
[Automation Exercise](https://automationexercise.com)

> You can replace this with any other demo website if you test on a different site.

- Validation Testing
- Positive & Negative Testing
- Regression Testing

## How to Run
1. Open the repository  
2. Refer to the **Test Plan** to understand scope  
3. Execute test cases in **Test_Cases** on the application under test  
4. Log defects in **Bug_Report**  
5. Summarize results in **Test_Summary**
